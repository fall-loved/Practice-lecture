Богович Юрий, варианты 10, 12, 4

Условия ДЗ

10. Создайте статический класс-репозиторий для хранения коллекции строк в памяти. Реализуйте в контроллере методы GET (вернуть все) и POST (добавить одну).
12. Создайте middleware, который добавляет кастомный заголовок X-Response-Time с текущей датой и временем в каждый исходящий ответ.
4. Создайте сервис IRequestCounter с методом для увеличения и получения счетчика. Зарегистрируйте его как Scoped. Внедрите его в два разных действия контроллера и убедитесь, что для одного запроса используется один экземпляр.

Запуск

Для запуска необходим .NET sdk версии 9.0.
Можно запускать как через консоль, так и через IDE.
Документация с возможностью тестировая Endpoint'ов находится по путь localhost:5000/swagger (при запуске через ide откроется автоматически)
