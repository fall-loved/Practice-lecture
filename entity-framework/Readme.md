Условие ДЗ

В проекта создать директорию Data, в ней необходимо описать наследника класса DbContext для работы с базой данных на основе ваших моделей по теме "Агентство недвижимости"

В корне проекта создать директорию Models, в которой будут размещены сущности хранимые в базе данных
Сущности: Property, Owner, Client, Contract, Viewing, Agent

Особенности: поиск по параметрам, история просмотров, договоры аренды/продажи
В примере работы описанно добавление сущностей в таблицу поиск сущности по параметрам с помощью LINQ-запросов, получение истории просмотров (сущностей Viewing для определённого клиента), добавлени и удаление договора (сущность Contract).


Запуск

Для запуска необходим .NET sdk версии 9.0 и docker (или СУБД postgres, но нужно менять порт в классе EstateContext)

Сначала необходимо поднять БД через docker (для этого есть compose-файл), затем применить миграции (dotnet tool run dotnet-ef database update)

Далее можно собирать и запускать проект любым удобным способом. Программа должна вывести список сущностей Property и список сущностей Viewing для пользователя, созданного при выполении программы
